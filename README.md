# ⚡ محاكي خوارزميات جدولة المعالج الاحترافي | CPU Scheduler Simulator Pro

تطبيق ويب تفاعلي متطور لمحاكاة وتحليل خوارزميات جدولة وحدة المعالجة المركزية (CPU Scheduling Algorithms) في نظم التشغيل. تم بناء هذا المشروع ليكون أداة تعليمية وعملية قوية تساعد الطلاب والباحثين على فهم وتصور كيفية إدارة المعالج للعمليات بشكل مرئي وديناميكي وفوري.

## ✨ المميزات الرئيسية (Features)
* 📊 **إدخال تفاعلي وديناميكي للبيانات:** جدول متطور (شبييه بـ Excel) يتيح إضافة، تعديل، وحذف العمليات (Process ID, Arrival Time, Burst Time) بعدد غير محدود وفي الوقت الفعلي.
* ⚙️ **دعم الخوارزميات الأساسية:**
  * خوارزمية القادم أولاً يُخدم أولاً `FCFS (First Come First Serve)`
  * خوارزمية العمل الأقصر أولاً `SJF (Shortest Job First) - Non-preemptive`
* 🔍 **نظام ذكي للتحقق من البيانات (Data Validation):** يمنع انهيار التطبيق عبر فحص المدخلات وتنبيه المستخدم في حال وجود حقول فارغة أو قيم غير صحيحة.
* 📉 **مخطط غانت الزمني (Gantt Chart):** تمثيل بصري عالي الدقة يعرض تسلسل تنفيذ العمليات على المعالج بألوان ديناميكية متناسقة مع الواجهات المظلمة (Dark Theme).
* ⏱️ **حساب مؤشرات الأداء (KPIs):** احتساب وعرض متوسط زمن الانتظار (Avg Waiting Time) ومتوسط زمن الإنجاز (Avg Turnaround Time) بدقة رياضية متناهية.

## 💻 التشغيل المحلي للمشروع (Local Setup)

إذا كنت ترغب في تشغيل المشروع على جهازك الشخصي، اتبع الخطوات التالية:

### 1. استنساخ المستودع (Clone the Repository)
```bash
git clone https://github.com
cd CPU-Scheduler-Simulator
```

### 2. تثبيت المكتبات المطلوبة (Install Dependencies)
تأكد من تثبيت الحزم اللازمة المذكورة في ملف `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 3. تشغيل التطبيق (Run the Application)
قم بتشغيل المحاكي عبر الأمر التالي:
```bash
streamlit run app.py
```
سيفتح التطبيق تلقائياً في متصفح الويب الخاص بك على الرابط المحلي `http://localhost:8501`.
